# Branding Direction

## Chosen Direction

The current preferred direction is **Orbital Ball**.

Why this direction won:

- it maps directly to the core interaction model: a virtual ball driving precise cursor movement
- it is broader than a single Apple Watch client
- it scales from tray glyph to app icon to marketing art
- it avoids looking like a generic cursor utility or a temporary AI promo illustration

## Visual Principles

- Base: deep graphite / midnight background
- Core mark: dark spherical ball
- Motion cue: electric cyan orbital arc
- Accent: restrained violet secondary accent
- Product feel: precise, technical, calm, not playful or toy-like

## Implementation Guidance

- Use the Orbital Ball symbol as the primary product mark.
- Avoid watch hardware, hands, or fingers in the core icon.
- Keep desktop and mobile icons in the same visual family.
- Reserve more illustrative assets for screenshots, README hero art, or store collateral.

## Current Repository Status

The watch trackball visual is being aligned to this direction first. App icons and marketing assets can be migrated next.
